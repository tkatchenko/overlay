# Image Overlay

A browser extension to overlay images on web pages.

## Features

- Control position, scale, opacity, and inversion of multiple images.
- Persists images and settings.
- Draggable UI panel for controls.
- LiveReload support for development.

## Usage

1.  Load the extension in your browser.
2.  Click the extension icon to open the controls.
3.  Upload an image.
4.  Adjust the image overlay using the controls.
